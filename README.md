# Workforce-Analytics-and-Attrition-Visualization-Dashboard
The HR Analytics Dashboard is an interactive Power BI project designed to analyze key workforce metrics and uncover insights about employee attrition, demographics, and compensation trends.
It helps HR teams and management make data-driven decisions to improve retention and workforce planning.

**Key Features**

1. Comprehensive KPI Tracking: Displays total employees, attrition count, attrition rate, average salary, and average age.

2. Attrition Analysis: Breaks down attrition by age group, education, salary slab, and years at the company.

3. Interactive Visuals: Dynamic slicers and filters for exploring data by department, gender, and job role.

4. DAX Measures: Custom calculations for attrition rate, numeric conversion of Yes/No attrition data, and trend analysis.

5. User-Friendly Interface: Professionally styled dashboard with clear, intuitive visuals for quick insights.


**Tech Stack & Tools**

Data Visualization: Power BI

Data Modeling: DAX (Data Analysis Expressions)

Data Cleaning & Preparation: MS Excel / Power Query

Other Tools: SQL, Google Sheets (for data sourcing)


**Key Insights**

Identified an overall attrition rate of 16.12% with higher turnover in the 26-35 age group.

Provided actionable insights for HR to improve retention strategies and optimize hiring decisions
